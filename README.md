#### 写在前面
<p style="font-size:12px;">考虑到部分朋友不知道如何使用git或者node这样的命令，我简化了使用方法，下载后直接运行`run.bat`就可以了。</p>

#### 系统准备
不需要任何准备,开箱即用!
但是最好你的系统要安装Chrome浏览器, 因为发现有空房间后, 会弹出浏览器提示.
